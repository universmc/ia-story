# Ressources d'Apprentissage sur le Développement Full-Stack

Dans cette section, vous trouverez des ressources utiles pour apprendre et travailler sur le développement full-stack de notre projet `ia-story`. Assurez-vous de consulter ces ressources pour approfondir vos connaissances et améliorer votre projet.

## Structure du Projet

- `/src` : Le répertoire source contient tous les fichiers front-end et back-end de notre application.

  - `/css/style.css` : Fichier CSS pour la stylisation de l'interface utilisateur.

  - `/js/script.js` : Fichier JavaScript principal pour les fonctionnalités front-end.

  - `/js/main.js` : Fichier JavaScript principal pour l'initialisation et les interactions front-end.

- `/srv` : Le répertoire serveur contient des fichiers liés à la partie back-end de notre application.

  - `/server.js` : Fichier principal du serveur Node.js pour la gestion des requêtes et des routes.

  - `/database.sql` : Script SQL pour la création et la gestion de la base de données.

## Ressources d'Apprentissage

Voici quelques ressources d'apprentissage recommandées pour le développement full-stack :

1. [Documentation MDN Web Docs](https://developer.mozilla.org/fr/docs/Web) : Une excellente ressource pour apprendre les technologies web, notamment HTML, CSS et JavaScript.

2. [Node.js Documentation](https://nodejs.org/fr/docs/) : La documentation officielle de Node.js pour apprendre à développer des applications back-end en JavaScript.

3. [Express.js Documentation](https://expressjs.com/fr/) : Si vous utilisez Express.js comme framework back-end, cette documentation est essentielle.

4. [SQL Tutorial](https://www.w3schools.com/sql/) : Pour apprendre les bases du langage SQL utilisé dans la gestion de bases de données.

5. [GitHub Pages Documentation](https://docs.github.com/fr/pages) : Si vous prévoyez de déployer votre interface utilisateur sur GitHub Pages, cette documentation vous guidera à travers le processus.

N'hésitez pas à explorer ces ressources pour renforcer vos compétences en développement full-stack. Si vous avez des questions ou besoin d'aide supplémentaire, n'hésitez pas à nous contacter.
